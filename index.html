My Projects:

  • YourVillage

    My Epicodus capstone project. The site uses .Net/Entity Framework Core, and is built as a C# MVC app with a mySQL database. The site is a communications platform for families and caregivers, using claims-based authorization to allow a user to create a secure Family profile with read-write access and then share read-only access with selected Caregivers, while also having Caregiver access to other authorized Family profiles. Cloud-hosted with DigitalOcean, utilizing an Ubuntu server and Caddy reverse-proxy server.

  • Earthly Remedies:

    An API created in C#, consumed by a React front end application. A database of user-generated traditional, natural and homeopathic remedies.

  • Endangered Animals Education Site:

    An education site consuming the IUCN RedList API, a huge API that required call cacheing and optimization in order to make the front end responsive. Front end uses C# and JavaScript to handle SVG files and custom CSS styling.

  • COVID Tracker Maps:

    A UI/UX-focused project in JavaScript consuming a major COVID-19 data API, as well as the US Census API. Data is displayed using Leaflet.js and Canvas.js. Please note that the project was created in April 2020, and does not reflect changes in the way COVID-19 data has been reported and interpreted since then.
